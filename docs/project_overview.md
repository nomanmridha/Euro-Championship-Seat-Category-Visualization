# Project Overview

## Title
European Championship Seat Category Data Visualization

## Academic Context
- University: South Westphalia University of Applied Sciences
- Course: Data Science / Data Visualization
- Project Type: Academic Data Analytics Project

## Project Description
This project analyzes seating category data from European Championship events.
The goal is to uncover pricing patterns, customer preferences, and revenue insights
using Tableau-based visual analytics.

## Deliverables
- Cleaned and processed datasets
- Tableau visualization dashboards
- Analytical insights report
- Business-oriented recommendations

## Learning Outcomes
- Practical experience with real-world datasets
- Advanced Tableau dashboard development
- Business-focused data storytelling
- Professional GitHub project documentation
