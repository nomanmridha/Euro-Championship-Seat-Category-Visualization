# Methodology

## Project Objective
The objective of this project is to analyze seat category pricing and customer behavior
during European Championship events using data visualization techniques.

## Data Sources
- Raw seat category dataset provided by university faculty
- Processed dataset after cleaning and transformation
- Tableau dashboards generated from processed data

## Data Preparation
- Removed missing and inconsistent pricing values
- Standardized seat category and country names
- Verified numerical ranges for ticket prices
- Ensured consistency between gender, device, and region attributes

## Visualization Approach
Tableau was used to:
- Identify pricing differences across regions and countries
- Analyze customer behavior by device and gender
- Compare average ticket prices across multiple dimensions

## Tools Used
- Microsoft Excel (data cleaning and preprocessing)
- Tableau Desktop (visual analytics)
- GitHub Markdown (documentation)

## Limitations
- Dataset is limited to academic use
- No real-time or transactional data
- Analysis focuses on descriptive insights
